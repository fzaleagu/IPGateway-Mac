IPGateway

IPGateway is a simple but useful app for logging in/out of the IP gateway of Peking University.

This is IPGateway for Mac running OS X.

IPGateway for iOS is available in the App Store here: https://itunes.apple.com/cn/app/ipgateway/id493761518?l=en&mt=8

IPGateway for iOS 's source code is hosted at: https://bitbucket.org/shengbinmeng
